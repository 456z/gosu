Gosu, What's That?

Gosu is a 2D game development library for the Ruby and C++ programming languages. Gosu features easy to use interfaces to:

  • Hardware accelerated 2D graphics
  • Sound samples and music in all formats supported by FMOD
  • Keyboard, mouse and gamepad/joystick input

Gosu's current version is 0.6.5 (0.6.4 on Windows, where no bug had to be fixed).

Platforms Supported by Gosu

Gosu currently supports four development platforms. Click on one of them to get installation details:

  • Ruby under Windows, Mac OS X and Linux
  • C++ using Microsoft Visual C++ 7.1 (not tested with 7.0, but might work)
  • C++ using Xcode under Mac OS X
  • C++ using gcc/g++ under Linux

Support for Microsoft Visual C++ 8.0 coming real soon!

Using Gosu with Apple Xcode

Installing the Gosu framework

Copy the supplied mac/Gosu.framework file to /Library/Frameworks/Gosu. This is the only step that is also necessary for users of your game.

Installing Other Libraries

Boost

This is a collection of libraries that is used throughout Gosu. Many parts of Boost have been marked for inclusion in the next standard of C++, so you should take a look at it anyway!

Download and extract the latest boost release. Open a terminal, go to that directory and run: 

    /.configure
    make
    sudo make install
    sudo ln -s /usr/local/include/boost-1_33_1/boost /usr/local/include/boost
(Replace "1_33_1" with boost's current version.)

Creating a Gosu project

In Xcode, create a new Cocoa application project. Remove Resources/MainMenu.nib, add a reference to the installed Gosu.framework and replace the main.m file by your game's code files - one of the tutorial files, for a start. (But don't forget to copy the tutorials' media folder into your project's build directory).
